# Glootie for JetBrains IDEs

AI state machine plugin for all JetBrains IDEs.

## Supported IDEs

- IntelliJ IDEA
- PyCharm
- WebStorm
- GoLand
- CLion
- Rider
- PhpStorm

## Installation

From Plugin Marketplace:
1. Open IDE
2. Settings → Plugins → Marketplace
3. Search "Glootie"
4. Install and restart

## Features

- Unified state machine
- Code analysis
- Real-time sync
- Hot reload
- Multi-language support

## Quick Start

1. Tools → Glootie → Activate
2. Ctrl+Alt+Shift+G (Show State)
3. Right-click → Glootie → Analyze

## Configuration

Settings → Tools → Glootie
